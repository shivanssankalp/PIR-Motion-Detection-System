# 🚶 PIR Motion Detection System using Arduino

## Overview

This project demonstrates a motion detection system using a PIR (Passive Infrared) sensor and Arduino. When motion is detected, an LED automatically turns ON. When no motion is detected, the LED remains OFF.

## Features

* Motion Detection using PIR Sensor
* Automatic LED Control
* Real-Time Monitoring
* Simple IoT Sensor-Based Simulation

## Components Used

* Arduino Uno
* PIR Sensor
* LED
* Resistor
* Jumper Wires

## Working Principle

The PIR sensor continuously monitors infrared radiation changes. When a moving object is detected, the sensor output becomes HIGH, causing the LED to turn ON. When no motion is detected, the LED turns OFF.

## Applications

* Security Systems
* Smart Lighting
* Intruder Detection
* Home Automation
